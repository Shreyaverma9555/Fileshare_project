# Fileshare_project
“This project is a secure file sharing system that allows users to upload, share, and download files with OTP-based authentication to ensure data security and controlled access.”
🟢 Introduction
This web-based application, developed using Python and Flask, provides a secure platform for users to upload, store, and share files across a network. The system employs login authentication and OTP-based email verification to ensure that only authorized users can access files. It offers a safe and efficient solution for file sharing, catering to students, organizations, and individuals.
🟢 Purpose of the Project
The primary objectives of this project are to:
*   Provide secure file sharing
*   Prevent unauthorized access
*   Enable controlled file access through OTP verification
*   Replace existing insecure file transfer methods
🟢 Problem Statement
Traditional file sharing methods, such as email or USB drives, present several issues:
*   Limited file size in email
*   Security risks, including unauthorized access
*   Lack of proper file tracking
*   Lack of centralized storage
🟢 Proposed Solution
The system allows users to:
*   Register and log in securely
*   Upload files to the server
*   Share files with specific users
*   Verify identity using OTP before downloading
*   Access files based on permissions
🟢 Technologies Used
*   Python → Backend logic
*   Flask → Web framework
*   HTML/CSS → Frontend interface
*   SQLite/MySQL → Database
*   SMTP (Gmail) → Sending OTP emails
🟢 Working of the System
*   User logs into the system.
*   After login, the user is redirected to the email verification page.
*   User enters their email to receive an OTP.
*   The system generates and sends an OTP via email.
*   User enters the OTP for verification.
*   If the OTP is correct, the user can access files.
*   User can upload, download, and share files.
